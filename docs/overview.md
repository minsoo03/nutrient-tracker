# 앱 개요

## 기본 정보
| 항목 | 내용 |
|------|------|
| 앱 이름 | 미정 |
| 플랫폼 | iOS / Android |
| 프레임워크 | Flutter |
| 백엔드 | Firebase (Firestore, Auth, Storage) |
| 영양소 DB | 식품안전처 + USDA FoodData Central |
| AI | OpenAI API (Phase 5) |

## 핵심 기능
- 음식·보충제·약물 입력 → 영양소 자동 집계
- 매크로 (탄수화물·단백질·지방·당·칼로리) 추적
- 장기 건강 지표 (신장·간·카페인) 추적
- 기준 초과 시 경고 알림

## 개발 로드맵
| Phase | 내용 | 기간 |
|-------|------|------|
| 1 | Flutter 세팅, Firebase 연결, 로그인/회원가입 | 2주 |
| 2 | 음식 검색·입력, 영양소 DB 연동, 일일 집계 | 4주 |
| 3 | 신장·간·카페인 트래킹, 경고 시스템 | 3주 |
| 4 | 대시보드 차트, 히스토리, 트렌드 그래프 | 3주 |
| 5 | AI 자연어 입력, 바코드 스캔, 사진 인식 | 4주 |
| 6 | 앱스토어 심사, 개인정보처리방침, 출시 | 2주 |
