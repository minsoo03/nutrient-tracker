# 앱 개요

## 앱 이름
미정 (가칭: Nutrient Tracker)

## 핵심 컨셉
사용자가 하루 섭취한 음식·음료·보충제·약물을 입력하면,
영양소 DB 기반으로 탄수화물/단백질/지방/당 등 매크로 영양소와
**신장·간에 영향을 주는 성분, 카페인**까지 종합 집계하여 시각적으로 보여주는 일일 건강 추적 앱.

## 차별점
일반 칼로리 트래커와 달리 **장기 건강 영향 지표**까지 추적 → 단순 다이어트를 넘어 진정한 건강 관리 도구

## 플랫폼 & 기술 스택
| 항목 | 내용 |
|------|------|
| 플랫폼 | iOS / Android |
| 프레임워크 | Flutter |
| 백엔드 & DB | Firebase (Firestore, Auth, Storage) |
| 영양소 DB | 식품안전처 DB + USDA FoodData Central API |
| AI/ML | OpenAI API (음식명 → 영양소 추정) |

## 개발 단계
| 단계 | 내용 | 기간 |

| Phase 6 | 앱스토어 심사, 개인정보처리방침, 출시 | 2주 |
